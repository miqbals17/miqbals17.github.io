Analisis Deret Fourier¶

Deret fourier adalah sebuah pendekatan suatu fungsi dengan menggunakan fungsi basis sinus dan cosinus.

Semua bentuk gelombang yang keseluruhannya terdefinisi pada bidang real dapat diaproksimasi dengan menggunakan deret fourier jika gelombang tersebut periodik.

Persamaan Deret Fourier $$f(x)=a_0+\sum_{n=1}^{N}[a_n\cos(\frac{n\pi}{L}x)+b_n\sin(\frac{n\pi}{L}x)]$$

dengan $$a_0=\frac{1}{2L}\int_{-L}^{L}f(x)dx$$ $$a_n=\frac{1}{L}\int_{-L}^{L}f(x)\cos(\frac{n\pi}{L}x)dx$$ $$b_n=\frac{1}{L}\int_{-L}^{L}f(x)\sin(\frac{n\pi}{L}x)dx$$

dan

$f(x)=$ Fungsi gelombang yang akan diaproksimasi

$P=2L=$ Periode Gelombang yang diaproksimasi

$n=1,2,3,...,\infty$ (Secara teori)

Karena square wave memiliki periode = $2\pi$, maka: $$a_0=\frac{1}{2\pi}\int_{-\pi}^{\pi}f(x)dx$$ $$a_n=\frac{1}{\pi}\int_{-\pi}^{\pi}f(x)\cos(nx)dx$$ $$b_n=\frac{1}{\pi}\int_{-\pi}^{\pi}f(x)\sin(nx)dx$$

dengan $$ f(x) = \left\{ \begin{array}\\ -1,&-\pi<x<0 \\ 1,&0<x<\pi \end{array} \right. $$

Perhitungan $a_n$ dan $b_n$¶

$$a_n=\frac{1}{\pi}(\int_{-\pi}^0-\cos(nx)dx + \int_0^{\pi}\cos(nx)dx)$$$$b_n=\frac{1}{\pi}(\int_{-\pi}^0-\sin(nx)dx + \int_0^{\pi}\sin(nx)dx)$$

def sum(N=0):
  integral_sum = 0
  for i in range(1, N):
    n = i
    an = 1/np.pi * (quad(an_1, -np.pi, 0, args=(n))[0] + quad(an_2, 0, np.pi, args=(n))[0])
    an = an*np.cos(i*x)
    bn = 1/np.pi * (quad(bn_1, -np.pi, 0, args=(n))[0] + quad(bn_2, 0, np.pi, args=(n))[0])
    bn = bn*np.sin(i*x)
    integral_sum = integral_sum + (an + bn)
  return integral_sum

#@title
x = np.arange(-10, 10, .01)

def int_neg(x, n):
  return (x)*np.sin(n*x)

def int_pos(x, n):
  return (x)*np.sin(n*x)

def sum(N):
  final = 0
  for i in range(1, N+1):
    n = i
    int_sum = quad(int_neg, -np.pi, 0, args=(n))[0] + quad(int_pos, 0, np.pi, args=(n))[0]
    summation = 1/np.pi * int_sum * np.sin(n*x)
    final = final + summation
  return final
